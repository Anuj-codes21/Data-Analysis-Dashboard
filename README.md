#  AV Store Sales Data Analysis (Interactive Dashboard using MS Excel)

##  Project Overview
This project presents a comprehensive sales data analysis for AV Store using Microsoft Excel.  
The objective was to design an interactive annual sales dashboard that enables business owners to monitor performance, understand customer behavior, and make data-driven decisions for future growth.

The dashboard transforms raw transactional data into meaningful insights through data cleaning, pivot tables, calculated metrics, and dynamic visualizations.

---

##  Business Objectives
- Analyze monthly sales and order trends  
- Identify top-performing states and regions  
- Compare purchasing behavior between male and female customers  
- Evaluate order status distribution  
- Determine best-performing sales channels  
- Identify highest-selling product categories  
- Measure overall order delivery performance  

---

##  Tools & Techniques Used
- Microsoft Excel  
- Data Cleaning & Preprocessing  
- Pivot Tables & Pivot Charts  
- Slicers for Interactive Filtering  
- Conditional Formatting  
- KPI Metrics Calculation  
- Data Visualization & Dashboard Design  

---

##  Dataset
The dataset contains transactional sales data including:
- Order Date
- Customer Gender & Age Group
- State
- Sales Channel
- Order Status
- Product Category
- Revenue & Order Count

---

##  Dashboard Features
- Monthly Sales vs Orders comparison chart  
- Gender-based sales contribution analysis  
- Top 10 states by total revenue  
- Channel-wise performance breakdown  
- Category-wise sales distribution  
- Order status tracking  
- Fully interactive slicers for dynamic analysis  

---

##  Data Analysis Process
1. Verified dataset for missing values and inconsistencies  
2. Standardized data formats and cleaned anomalies  
3. Created pivot tables based on defined KPIs  
4. Designed calculated metrics for performance tracking  
5. Combined all pivot visuals into a single interactive dashboard  
6. Applied slicers for real-time filtering and user interaction  

---

##  Key Insights
- Female customers contributed approximately 65% of total sales  
- Maharashtra and Karnataka were among the top-performing states  
- Certain sales channels generated significantly higher revenue  
- Specific product categories consistently outperformed others  
- Majority of orders were successfully delivered, indicating strong fulfillment performance  

---

##  Business Impact
The dashboard provides clear visibility into sales trends and customer behavior, enabling management to:
- Optimize marketing strategies  
- Focus on high-performing regions and categories  
- Improve channel performance  
- Support strategic decision-making with real-time insights
